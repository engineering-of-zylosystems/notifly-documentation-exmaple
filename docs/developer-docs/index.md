## 시작하기

1. **플랫폼 선택**: 앱 또는 웹사이트에 맞는 SDK를 선택하세요
2. **SDK 설치**: 선택한 플랫폼의 설치 가이드를 따라하세요
3. **초기화**: 프로젝트 ID로 SDK를 초기화하세요
4. **테스트**: 연동 테스트를 통해 올바르게 작동하는지 확인하세요

## 지원 플랫폼

- **모바일**: iOS, Android, React Native, Flutter
- **웹**: JavaScript, Google Tag Manager
- **서버**: HTTP API
- **테스트**: 연동 테스트 도구
