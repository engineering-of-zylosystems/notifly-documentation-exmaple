# 유저 여정

유저 여정은 고객의 행동과 속성에 따라 메시지를 자동으로 발송할 수 있는 **마케팅 자동화 기능**입니다.  
분기 조건, 일정 대기, 다양한 채널 발송을 하나의 시나리오로 연결해 운영할 수 있으며, 이를 통해 더 효과적인 상호작용과 향상된 고객 경험, 높은 전환율을 달성할 수 있습니다.
