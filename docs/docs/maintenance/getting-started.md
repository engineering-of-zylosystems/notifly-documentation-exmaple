## 유저 여정이란?

유저 여정은 사용자가 앱이나 웹사이트에서 경험하는 일련의 단계들을 의미합니다. 노티플라이를 통해 각 단계에서 적절한 알림을 발송하여 사용자 경험을 향상시킬 수 있습니다.

## 주요 기능

- **자동화된 메시지 발송**: 사용자 행동에 따른 자동 알림
- **개인화**: 사용자별 맞춤 메시지
- **다채널 지원**: 푸시, 이메일, SMS 등 다양한 채널
- **실시간 분석**: 캠페인 성과 추적
